# 🌲 Escape the Forest 🌲

A terrifying 3D first-person horror game where you must escape through the dark forest while being hunted by a monstrous creature!

## 🎮 Game Features

✨ **Scary 3D Environment**
- Dark, immersive forest atmosphere
- Dense trees and ambient lighting
- Fog and spooky sound effects

👹 **The Monster**
- Gray creature with long, eerie arms
- Intelligent AI that hunts you down
- Gets faster when it detects you
- Red glowing eyes that track your movement

🏃 **Intense Gameplay**
- Sprint mechanic with limited stamina
- Monster proximity warning system
- Smooth first-person camera controls
- Realistic physics

🎯 **Objectives**
- Reach the yellow glowing gate to escape
- Avoid being caught by the monster
- Manage your stamina wisely

## 📱 Mobile & Desktop Play

**Option 1: Play Online (EASIEST)**
1. Go to: https://Aidweenie.github.io/My-game-/
2. Click to start playing immediately!

*Note: If that link doesn't work yet, enable GitHub Pages in your repo settings*

**Option 2: Local File**
1. Download the repository (or clone it)
2. Open `index.html` directly in your browser
3. The game will load instantly!

### 🕹️ Controls

**Desktop:**
- **WASD** - Move around
- **Mouse** - Look around
- **SPACE** - Sprint (uses stamina)
- **Click Mouse** - Lock/unlock mouse
- **ESC** - Release mouse lock

**Mobile:**
- **Left Joystick** - Move forward/backward/strafe
- **Right Joystick** - Look around
- **SPRINT Button** - Tap and hold to run

## 📊 UI Elements

- **Stamina Bar** (top left) - Your sprint energy
- **Monster Proximity Meter** (top right) - How close the monster is (red = DANGER!)
- **Yellow Gate** (ahead) - Your escape route
- **Crosshair** (center) - For aiming

## 💻 Technical Details

Built with:
- **Three.js** - 3D graphics library
- **WebGL** - Hardware-accelerated graphics
- **Vanilla JavaScript** - Game logic and AI

## 🎨 Graphics

- Procedurally generated forest with 60+ trees
- Dynamic lighting and shadows
- Screen shake effects when the monster is near
- Red vignette effect as danger increases
- Realistic camera movement

## 🤖 Monster AI

The monster features:
- Distance-based detection system
- Intelligent pathfinding toward the player
- Animation system for arms when chasing
- Patrol behavior when idle
- Adaptive difficulty (speeds up when chasing)

## 🎵 Atmosphere

The game creates tension through:
- Dark color palette
- Fog effects reducing visibility
- Proximity warnings
- Visual feedback when danger is near
- Screen distortion when the monster is close
